Welcome to Sylva importation manager. This tool will allow you to upload plain CSV information to Sylva Graph Format. You can learn more about CSV here.

Sylva supports Unicode (UTF-8) CSV files using several column separators and text delimiters.

You, or anyone on your team, should had created the graph database with its schema before using this tool. If you want to learn more about schemas and databases creation you can find information here.

After selecting the target database, select a file from your computer and press Next.
